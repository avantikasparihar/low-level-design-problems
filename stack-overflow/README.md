### Requirements:

1. User is able to create posts with their questions
2. User is able to add comments in the posts
3. User is able to reply to the comments in the posts
4. User can delete a post they created
5. User can update / edit the contents of their posts / comments
6. User can upvote / downvote any post(the vote counter should only increase / decrease by one per user)
7. User can upvote / downvote any comment
8. User gain / lose points for receiving votes on their content
9. User receives badges based on the number of votes or activity
10. Post can have different statuses like "open", "answered", "stale", etc
11. Moderators can remove posts / comments
12. Concurrency should be handled for the cases where more than one users try to upvote simultaneously
