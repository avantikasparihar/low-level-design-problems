### Requirements:

1. User is able to list movies that are available to book.
2. User is able to book a movie that is scheduled in a future time and capacity is available.
3. User is able to book more than one tickets to the same movie.
4. Theatre admin is able to add movie schedules in the system, define the schedule, capacity, etc.
5. Theatre admin is able to remove the movies that were added.
6. Users should be able to concurrently book tickets in the same movie / theatre.
